介紹一個基本的導覽

在 Tocas Xiaoan 中，所有功能皆有輸出在網頁中及輸出在 console 裡的版本，若您要輸出在 console 中，在後方加上 _con 即可

原始碼
//底下程式碼會輸出在網頁上
xiaoan.earlySpring404()
//底下程式碼會輸出在 console 裡
xiaoan.earlySpring404_con()


404 找不到始春輸出一個找不到始春的錯誤！網頁中會以一個 Tocas UI 的訊息框輸出，而且您可以指定樣式，樣式請參閱 Tocas UI

原始碼
xiaoan.earlySpring404()
xiaoan.earlySpring404("inverted warning")
